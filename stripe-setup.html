「買える状態」にする最後の手順

サイトはもう完成しています。あとは Stripe（決済サービス）とつなぐだけ。所要 30分ほどです。

1

Stripe アカウントを作る（無料）

stripe.com/jp にアクセスし、メールアドレスで登録します。月額・初期費用は0円。手数料は決済ごとに 3.6% だけです。

初期費用 ¥0
月額 ¥0
決済手数料 3.6%
本人確認（氏名・住所・振込先の銀行口座）が必要です。売上はこの口座に自動で振り込まれます。
2

商品ごとに「決済リンク」を作る

Stripe管理画面 → 左メニューの「支払い」→「Payment Links（支払いリンク）」→「新規作成」。商品名と価格を入れるだけで、支払いページのURLが発行されます。

  • 商品は全部で6つ（飯碗・湯呑・小皿・マグ・中鉢・七寸皿）
  • 各リンクの設定で 「配送先住所を収集する」をオンにしておく（発送に必要）
  • できたURLは https://buy.stripe.com/xxxx の形です
6商品ぶん、6本のURLができます。これをメモしておいてください。
3

URLをサイトに貼る（これだけ）

index.html の上のほうにある商品リストの、各商品の stripe:'' の '' の中にURLを貼ります。

const PRODUCTS = [
  { name:'粉引 飯碗', price:'¥3,200', … stripe:'https://buy.stripe.com/aaaa' },
  { name:'灰釉 湯呑', price:'¥2,400', … stripe:'https://buy.stripe.com/bbbb' },
  …
];
これで「購入する」ボタンが本物の決済ページに飛ぶようになり、買える状態の完成です。（貼るまでは「販売準備中です」と表示され、誤購入は起きません）
4

特定商取引法ページを記入

tokushoho.html の オレンジ色の箇所（氏名・住所・電話番号など）を、実際の情報に書き換えます。ネット販売では法律で必須です。

5

公開（反映）

変更を保存して git push すれば、同じURLに自動で反映されます。この作業は私（Claude）がやります。「Stripeリンクできたよ」と言ってもらえれば、貼り付けから公開までまとめて対応します。

まとめ：あなたがやるのは「①Stripe登録 → ②6本のリンク作成」だけ。あとの貼り付け・特商法・公開は私に投げてOKです。
